# storage AGENTS

## 文件约定

- 运行 `just` 查看所有可用命令（Go 工具链：`just go ...`）
- 项目背景和模块索引见 `README.md` 和 `docs/`
- 行为约束见本文档

## 开发环境

- Go ≥ 1.26（见 `go.mod`）
- 无需额外初始化，直接 `just go test`、`just go gen` 等入口

## 编码风格

- 文档、注释、错误和测试说明使用中文，专有名词除外
- 先做最小盘点再开始改动
- 不做无关功能扩展、顺手重构或批量风格清理
- 不因目录已存在而默认新增模块、入口或注册关系
- 不回滚或覆盖自己未引入的变更，除非得到明确确认
- 优先保持既有架构分层，通过参数或依赖注入传递上下文
- 能完成最小验证时先验证再交付；无法验证时说明缺口

## 什么时候停、什么时候问

- 需求边界不清且存在多种互斥实现时停下
- 需要新增模块、入口或执行面时先确认
- 发现结构异常或控制面职责混写时先指出问题
- 需要执行高风险操作或引入外部凭据时请求人工处理

## 规则生效

- 只有落盘到 `AGENTS.md` 的规则才生效
- 子目录 `AGENTS.md` 只覆盖确有必要的局部规则，未覆盖的继承上级
